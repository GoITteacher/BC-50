# Задачі

## Задача №1

```
Для першої фоми, застосувати один прослуховувач події який зберігає введені користувачем дані у LS. Необхідно використовувати lodash, щоб дані збергіались не частіше ніж раз в секунду. Бібліотеку Lodash необзідно самостійно встоновити. Автоматично відновлювати значення полів після оновлення сторінки. Значення інпутів в LS потрібно зберігати окремо один від одного (тобто під різними ключами).
```

## Задача №2

```
Для другої форми, автоматично зберігати введені користувачем дані в LS. При оновленні сторінки відновити значення інпутів. Дані в LS повинні зберігатись у вигляді одного об'єкта.
```

## Задача №3

```
Для третьої форми, зберігати введені користувачем дані в LS при настиканні кнопки Save. Відновити дані у формі після натискання кнопки Load. Дані в LS повинні зберігатись у вигляді одного об'єкта.
```

## Задача №4

```
Задача полягає в написанні коду на JavaScript для створення головного класу "Person" та двох дочірніх класів "Student" та "Teacher".

Клас "Person" буде містити загальну інформацію про людину, таку як ім'я, прізвище, дату народження, стать та контактну інформацію. Клас "Person" також буде мати методи для встановлення та отримання цих властивостей.

Клас "Student" буде містити додаткову інформацію про студента, таку як назва його спеціальності та номер студентського квитка. Клас "Student" також матиме методи для встановлення та отримання цих властивостей.

Клас "Teacher" буде містити додаткову інформацію про викладача, таку як предмети, які він викладає та його рейтинг як викладача. Клас "Teacher" також матиме методи для встановлення та отримання цих властивостей.

Класи "Student" та "Teacher" наслідуватимуть властивості та методи класу "Person", тобто у них будуть ті ж самі властивості та методи, що й у класу "Person", але також матимуть власні додаткові властивості та методи, що відрізнятимуть їх від класу "Person".
```
