# Задачі

## Задача №1
```
Для першої фоми, застосувати один прослуховувач події який зберігає введені користувачем дані у LS. Необхідно використовувати lodash, щоб дані збергіались не частіше ніж раз в секунду. Бібліотеку Lodash необзідно самостійно встоновити. Автоматично відновлювати значення полів після оновлення сторінки. Значення інпутів в LS потрібно зберігати окремо один від одного (тобто під різними ключами).
```
## Задача №2
```
Для другої форми, автоматично зберігати введені користувачем дані в LS. При оновленні сторінки відновити значення інпутів. Дані в LS повинні зберігатись у вигляді одного об'єкта.
```
## Задача №3
```
Для третьої форми, зберігати введені користувачем дані в LS при настиканні кнопки Save. Відновити дані у формі після натискання кнопки Load. Дані в LS повинні зберігатись у вигляді одного об'єкта.
```
